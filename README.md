# Spring Boot Training

## Getting Started

### Prerequisites

- Java 11
- Gradle
- Docker
- Docker Compose

- Clone the repo

```sh
git clone https://github.com/luisramilian95/luis-ramirez-spring-training.git
```

## Task 01

A Hello World application using Spring-boot.

```sh
cd task01 && gradle clean bootRun
```

## Task 02
